## Allows older Minecraft client versions to connect to newer server versions.

# MultiVersion
A Multi Version plugin for PocketMine-MP Betteraltay (API3). This plugin is currently in development.

## Supported Versions
- 1.21.93 (`819`)
- 1.21.90 (`818`)
- 1.21.80 (`800`)
- 1.21.70 (`786`)
- 1.21.60 (`776`)
- 1.21.50 (`766`)
- 1.21.40 (`748`)
- 1.21.30 (`729`)
- 1.21.20 (`712`)
- 1.21.2 (`686`)
- 1.21.0 (`685`)
- 1.20.80 (`671`)
- 1.20.70 (`662`)
- 1.20.60 (`649`)
- 1.20.50 (`630`)
- 1.16.220-1.17.40
